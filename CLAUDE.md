# Project: leeflannery.com

Next.js 16 app with React 19, Tailwind CSS 4, and shadcn/ui. Deployed on Vercel with Bun.

## Tooling

- Use `bun` for everything: `bun install`, `bun run dev`, `bun run build`, `bun test`
- Do not use npm, yarn, pnpm, or node directly

## Git & Commits

- Do not add "Co-Authored-By" lines to commits
- Write concise commit messages in imperative mood
- Do not push unless explicitly asked
- Do not commit unless explicitly asked

## Code Style

- Follow existing patterns and conventions in the codebase
- Use TypeScript; avoid `any` types
- Prefer named exports
- Keep components small and focused
- Use Tailwind classes; avoid inline styles and CSS modules
